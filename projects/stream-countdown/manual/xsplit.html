Stream Counter instructions for Xsplit

Make sure you are using the latest Xsplit Broadcaster

  1. In Xsplit, select the Scene you want to display the countdown in.
  2. Then create a new Other > Webpage URL source
  3. Switch to https://

I have already created a countdown

  1. After you have clicked Create on the countdown page, copy the URL (Adress) of the page
  2. Paste the URL into the empty input box in Xsplit
  3. Click OK
  4. You can now resize the source to fit your stream. Scroll down for instructions on how to work with a green background or text

I have not yet created a countdown

  1. Steps for creating a countdown are always the same. See the OBS manual.
  2. Paste the URL into the empty input box in Xsplit
  3. Click OK
  4. You can now resize the source to fit your stream. Scroll down for instructions on how to work with a green background or text

Work with green background and text

  1. It doesn't matter if you made your text or your background green, the steps are the same
  2. Right Click the Countdown source
  3. Switch to the Color tab
  4. In the Keying section, switch to Chroma Key
  5. Make sure Key Color is switched to Green
  6. Feel free to play with the settings until you find the ones that look best for you
